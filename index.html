Python

Python é uma linguagem de programação de alto nível, interpretada e de propósito geral. Ela foi criada com ênfase na legibilidade do código e na simplicidade, tornando-a uma das linguagens mais populares para desenvolvimento de software.

PySimpleGUI

PySimpleGUI é uma biblioteca Python que facilita a criação de interfaces gráficas de usuário (GUIs) de forma rápida e simples. Ela é projetada para ser fácil de aprender e usar, permitindo que desenvolvedores criem interfaces interativas para suas aplicações de maneira eficiente.
Visitar site

SQLite3

SQLite3 é uma biblioteca de software que implementa um sistema de gerenciamento de banco de dados relacional embutido em uma única biblioteca. Ela fornece uma maneira simples e eficiente de armazenar, gerenciar e acessar dados em um banco de dados local, sem a necessidade de um servidor de banco de dados separado.
Visitar site

Pillow

Pillow é uma biblioteca Python, baseada na biblioteca PIL (Python Imaging Library), amplamente utilizada para manipulação de imagens. Ela fornece uma ampla gama de recursos e funcionalidades para abrir, editar, processar e salvar imagens de diversos formatos.
Visitar site

Java

Java é uma linguagem de programação de propósito geral e orientada a objetos amplamente utilizada. Ela foi desenvolvida para ser portável, robusta e segura, tornando-a uma das linguagens mais populares para desenvolvimento de software. Uma das principais características do Java é sua máquina virtual Java (JVM), que permite que os programas escritos em Java sejam executados em diferentes plataformas sem a necessidade de recompilação.
Visitar site

AWT

AWT (Abstract Window Toolkit) é uma biblioteca de componentes gráficos para desenvolvimento de interfaces de usuário em Java. Ela fornece um conjunto de classes e métodos para criar janelas, botões, caixas de texto, menus e outros elementos de interface gráfica.
Visitar site

HTML

HTML (Hypertext Markup Language) é uma linguagem de marcação utilizada para estruturar e exibir o conteúdo de páginas da web. Ela é a base fundamental da internet e permite a criação de documentos estruturados por meio de tags, que definem o significado e a organização dos elementos presentes em uma página.
Visitar site

CSS

CSS (Cascading Style Sheets) é uma linguagem de estilo utilizada para definir a aparência visual e o layout de elementos em páginas da web. Ele permite que os desenvolvedores apliquem estilos como cores, fontes, tamanhos e posicionamentos aos elementos HTML, proporcionando uma experiência visual consistente e atraente aos usuários.
Visitar site

Bootstrap

Bootstrap é um framework de desenvolvimento front-end popular e de código aberto, utilizado para criar sites e aplicativos web responsivos e visualmente atraentes. Ele oferece um conjunto abrangente de componentes pré-projetados, como botões, menus, formulários e carrosséis, além de uma grade flexível que facilita a criação de layouts responsivos.
Visitar site

JavaScript

JavaScript é uma linguagem de programação versátil amplamente utilizada para criar interatividade em páginas da web, permitindo o desenvolvimento de aplicativos web dinâmicos e responsivos, além de ser utilizado no desenvolvimento de aplicativos de servidor e até mesmo em aplicativos de desktop.
Visitar site

TypeScript

TypeScript é uma linguagem de programação de código aberto desenvolvida pela Microsoft. É um superconjunto sintático estrito de JavaScript e adiciona tipagem estática opcional à linguagem.
Visitar site

React

React é uma biblioteca JavaScript de código aberto desenvolvida pelo Facebook para o desenvolvimento de interfaces de usuário (UI) interativas e dinâmicas, oferecendo uma abordagem declarativa para a construção de componentes reutilizáveis e simplificando o desenvolvimento e manutenção de grandes projetos com sua arquitetura modular e eficiente.
Visitar site

Zustand

Zustand é uma biblioteca leve de gerenciamento de estado para React, oferecendo uma alternativa simples ao Redux. Permite compartilhar e manipular o estado de forma fácil e transparente, sem configurações complexas. Com recursos como devtools integrados, é uma escolha popular para gerenciamento de estado em projetos React de todos os tamanhos.
Visitar site

Expo

O Expo é um framework de desenvolvimento de aplicativos móveis que simplifica o processo de criação para iOS e Android. Ele oferece uma variedade de ferramentas e APIs pré-construídas para acelerar o desenvolvimento, permitindo que os desenvolvedores foquem na criação de experiências de usuário de alta qualidade usando JavaScript e React Native. Com tarefas como compilação de código e empacotamento automatizadas, o Expo torna o desenvolvimento mais rápido e acessível para desenvolvedores de todos os níveis de habilidade.
Visitar site

React Native

React Native é um framework de código aberto que permite desenvolver aplicativos móveis nativos para iOS e Android usando JavaScript e React. Com ele, é possível criar interfaces de usuário responsivas e ricas, aproveitando as funcionalidades nativas dos dispositivos. Sua capacidade de hot-reloading agiliza o desenvolvimento, tornando-o uma escolha popular para aplicativos móveis modernos e escaláveis.
Visitar site

Json Server

O JSON Server é uma ferramenta de código aberto que permite simular rapidamente uma API RESTful usando um arquivo JSON como fonte de dados. É amplamente utilizado para desenvolver e testar aplicativos front-end sem a necessidade de um back-end real, facilitando o processo de prototipagem e teste durante o desenvolvimento inicial de um projeto. Sua simplicidade e flexibilidade tornam-no uma escolha popular entre os desenvolvedores para criar APIs fictícias durante o desenvolvimento de aplicativos web e móveis.
Visitar site

Json Server Auth

O JSON Server Auth é uma extensão do JSON Server que adiciona suporte à autenticação de usuários usando tokens JWT. Ele permite simular um back-end com endpoints seguros para testar e desenvolver aplicativos que requerem autenticação, sem a necessidade de um servidor real. Essa extensão simplifica o desenvolvimento e teste de aplicativos front-end que exigem autenticação de usuários, permitindo aos desenvolvedores criar protótipos de forma rápida e eficiente.
Visitar site
